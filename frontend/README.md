# AI Tutor Frontend

The frontend is a Next.js application for course management, file selection, tutoring sessions, account pages, and educational video workflows.

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev -- -p 5173
```

Open http://localhost:5173 in a browser.

## Environment Variables

```env
NEXT_PUBLIC_API_URL=
```
